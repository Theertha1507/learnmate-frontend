# LearnMate — Frontend

**Agentic AI for Personalized Course Pathways** — IBM AICTE Problem Statement No. 12

This repo is the React frontend for LearnMate — an agentic AI coach that assesses a student's skill level and stated goals, then generates a personalized, adaptive course roadmap powered by **IBM Granite** via **IBM watsonx.ai**.

Live app: (https://learnmate-frontend-delta.vercel.app/)
Backend repo: [learnmate-backend](https://github.com/Theertha1507/learnmate-backend)

---

## What it does

- **Track selection** — choose an interest (Frontend Development, Cybersecurity, UI/UX Design, Backend Development)
- **Skill assessment** — a short, scored quiz classifies the student as beginner / intermediate / advanced *before* any recommendation is generated
- **Stated goal input** — a free-text field lets the student describe their goal (e.g. "placement ready in 3 months"), which genuinely changes which courses get recommended, not just how they're described
- **Live roadmap dashboard** — displays the agent's recommended courses, its guidance (generated by Granite), and lets the student mark courses complete
- **Real replanning** — marking a course complete calls the backend again, which re-runs the full agent loop and returns an updated roadmap

## Tech stack

- **React** (Create React App)
- **IBM Plex Sans / IBM Plex Mono / Unbounded** — typography
- Talks to the [LearnMate backend](https://github.com/Theertha1507/learnmate-backend) over REST, which handles all IBM watsonx.ai / Granite calls

## Setup

1. Clone this repo and install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file pointing at your backend:
   ```
   REACT_APP_API_BASE=http://localhost:5050
   ```
   (Use your deployed backend URL instead of localhost once the backend is deployed — e.g. `https://learnmate-backend-1.onrender.com`)

3. Run the app:
   ```bash
   npm start
   ```
   Opens at `http://localhost:3000`

> **Note:** This frontend requires the [LearnMate backend](https://github.com/Theertha1507/learnmate-backend) to be running (locally or deployed) to function — it has no standalone logic of its own.

## Deployment

Deployed on **Vercel**. The `REACT_APP_API_BASE` environment variable is set in the Vercel dashboard to point at the live backend URL.

## Design

The UI is intentionally built as a dashboard (sidebar navigation + data table) rather than a marketing-style layout, since this is a working tool a student would actually use repeatedly, not a landing page. Colors and typography choices are documented in `CoursePathwayAgent.css`.
